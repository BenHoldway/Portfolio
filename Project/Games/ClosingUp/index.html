Closing Up is a top down pixelated game which simulates a mundane job in a game environment.

Your job is simply to clean and tidy up the office space after a work day. Complete tasks, from cleaning to turning electricals off, to gain money. Use these funds to support your family and keep them healthy, you don't want them to fall ill.

❮ ❯

Project Development

This somewhat larger prototype was developed in a 6 week project sprint, created in the same module as "Chasm", which we were tasked with simulating a mundane job and "gamify-ing" it.

I had a limited time to create this game, which meant that I had to limit the number of mechanics that I would work on. Many mechanics, like different events that would change how the player played, or that gave the player different things to do as the game progressed, was outside of the scope for what the game ended up being.

This module and game taught me a lot of good things, from effective project and time management, as well as good ways to structure and track the development of the project. All these skills made the creation of this game a lot smoother and allowed for me to react when development got slightly behind more successfully.

I did make some mistakes in this project while trying to improve my code efficiency, however, which I have since learnt from and expanded my knowledge on more efficient Unity C# paradigms. This project specifically helped me grow a lot as a programmer and really showed me what my strengths and weaknesses are.
