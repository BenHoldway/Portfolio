    • Hidden Secrets (Nov, 2024)

    Hidden Secrets is a horror survival game, which the player must survive from an AI robot stalking them down. If they are caught and killed, a life will be taken away from a total of 3 lives.

    While avoiding the enemy AI robot, the player must keep systems active to stay alive. Oxygen is the most important, leading to a failed run if left unchecked. Power however simply controls the lights, and will make the game a lot darker if left. To restore or save these systems, the player must venture out into the map to search for the boxes in which control these systems.

    Survive the full 6 hours, keep the systems alive and don't let the robot see you!

    Team Size: Solo

    Duration: 8 Weeks

    Engine: UE5 (C++)

    Battle Of Talokov was a game, developed in Unreal Engine 5.1, in my first university year. The game started from a template project which had all of the assets I needed. The purpose of this project was learning about Game Mechanics, and efficient ways of implementing them.

    The module had me develop a level of a 3D top down tank game. I decided to build a level which had a story behind it; with the player infiltrating an enemy base to destroy the core power of the facility. Along the level, the player had a range of unique weapons at their disposal to find and use, to fight a bunch of different enemies.

    While I had some experience with using Unreal Engine beforehand, this was the first real project I worked on in the engine. I was able to effectively build base classes for different enemies and weapons, and create child classes of these to build unique utilities and objects throughout the game. With these bases, the game could be further developed and expanded on.
