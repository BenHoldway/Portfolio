Chasm is a very short 2D platformer, of which the player must parkour their way through the world to the end point.

Using a range of mechanics like double jump, wall jumping and usage of a grapple gun, the player traverses through multiple "rooms" of the world to avoid obstacles, collect coins and complete the level as fast as possible.

Project Development

This small prototype was developed in a 3 week project sprint, which had the prompt of "1 analog stick and 1 button, for multiple mechanics". It was the first of two games developed for a prototyping module, and created in Unity.

The project itself is very simple, due to the time constraint we had. While a game loop was made, the main point of this module was to create as many mechanics as we could with the very limited controls from the prompt.

The usage of the same controls in many situations, like jumping when on the ground to mid-air to against the wall, will change how the player will move, making the most out of a single key.
