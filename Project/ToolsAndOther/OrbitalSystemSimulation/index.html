This is an orbital simulation of a group of astronomical bodies, using DirectX v1.0 and C++. A number of properties is simulated, from different orbital speeds and radii, body sizes and rotation speeds as well.

The user is able to spectate each body using a navigation UI element, or switch between each using keys. While spectating, the user is able to zoom in and out to get a better view of the body, or viewing it from a wider shot.

Project Development

At the end of my first year, we had a module called Graphics Programming. This module was focused around teaching how to create objects in a world space, from the use of many components like vertices, colours and light.

From the given 9 weeks of learning about DirectX, and using C++ knowledge, I was able to create a planetary orbiting system that included a central star and many planets. Each object had a movement speed, rotation, orbit radius, emitting light and textures added to them to create this simulation.

While there is not much interactivity in this, and code had to be modified to change anything in the simulation, this is the building blocks for what could become more complex and customisable from the user's point of view.
