    Unity Dialogue Node Graph

    A custom tool, in Unity, which gives the user access to a graph view for setting up an easy Dialogue System for their games. With many features included, an easy node system can be created and linked into the game.

    This is still a work in process project, and is still in early development. With some time, I am hoping that this can be a dynamic tool for which I can use across multiple game projects, and potentially release for others to utilise as well.

    Functionaly:

    • Drag Navigation
    • Zooming
    • Node Selecting
    • Search Window to Add Element
    • Different types of nodes
    • Saving and Loading
    • Addons like a Blackboard and Minimap

    This is a custom tool that I started looking into, outside of University. I wanted to create something that I could use across multiple projects, whenever I needed dialogue.

    While also making a tool that I could use later on, it gave me a chance to look into Unity's GraphView system, as well as test out using StyleSheets and CSS. This has given me insight into how we can customise tools to our advantage, that helps us out when developing games.

    This is an ongoing side project for me, within my [Game Yet To Be Named] project. It has a lot of functionalities, as well as future functionalities planned, that lets the user make an intuitive dialogue system for their games.
