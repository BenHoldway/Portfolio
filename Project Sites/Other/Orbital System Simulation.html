DirectX Planetary System Simulation

At the end of my first year, we had a module called Graphics Programming. This module was focused around teaching how to create objects in a world space, from the use of many components like vertices, colours and light.

From 8 weeks of learning about DirectX, and using C++ knowledge, I was able to create a planetary orbiting system that included a central star and many planets. Each object had a movement speed, rotation, orbit radius, emitting light and textures added to them to create this simulation.

While there wasn't much interactivity in this, and code had to be modified to change anything in the simulation, this is the building blocks for what could become more complex and customisable from the user's point of view.

Programs Used: 

   - Visual Studio 2022
   - DirectX v1.0
